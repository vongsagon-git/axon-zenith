# 🧘 AXON ENLIGHTEN PROTOCOL (ตรัสรู้)

## Mission
**ตรัสรู้ไป ทำไป สร้าง MAP ไป** — ไม่ต้องวางแผนล่วงหน้า — กลับมาทำต่อได้ด้วย `/axon:ignite`
